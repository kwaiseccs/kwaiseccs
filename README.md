该工具帮助开发者检测代码依赖的安全问题，轻松识别代码中使用了哪些存在安全缺陷的开源组件，通过准确的修复方案和一键修复功能，快速解决安全问题。

### 支持功能

- 检测代码中引入的缺陷组件
- 查看缺陷组件的安全漏洞信息
- 提供修复方案

### 您也可以安装 JetBrains IDE 插件，实时检测您的代码

- 打开 JetBrains IntelliJ IDE 并进入插件市场
- 在插件市场中搜索 “murphysec”，查看详情并安装
